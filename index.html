Architecture

Multi-Agent Pipeline

Six specialized AI agents work in sequence to find, exploit, fix, and verify prompt injection vulnerabilities.

01

Clone PR

User triggers /scan via Discord. The bot clones the target GitHub Pull Request repository.

02

Scan Sinks

Identifies all LLM API call sites — OpenAI, Anthropic, and similar model invocation patterns.

03

Trace Flow

Follows data from user inputs through the code to model prompts, mapping injection vectors.

04

Red Team

Generates adversarial payloads to confirm exploitability of each discovered vulnerability.

05

Fix & Verify

Suggests and applies input sanitization. Verifies the patch neutralizes the attack vector.

06

Report

Posts a detailed security report to Discord with Approve / Reject buttons for human review.

Capabilities

Why Injection Sentinel?

Purpose-built to catch the vulnerabilities static analyzers miss.

Deep Injection Detection

Goes beyond pattern matching — traces actual data flow from user input to LLM prompts to find real injection vectors.

Adversarial Red Teaming

An AI agent generates real malicious payloads to prove exploitability — no theoretical risks, only confirmed vulnerabilities.

Automated Remediation

Doesn't just report — applies fixes, verifies the patch, and presents diffs for human-in-the-loop approval.

Discord-Native

Integrated as a slash command bot. Trigger scans, review reports, and approve fixes without leaving your team's workflow.

PR-Level Scanning

Scan any GitHub Pull Request URL. Catches vulnerabilities before they're merged into your production branch.

Multi-Agent Architecture

Six specialized agents — scanner, tracer, red-teamer, fixer, verifier, and reporter — each optimized for their role.

Quick Start

Get Running in Minutes

Four simple steps to deploy your own Injection Sentinel instance.

1

Clone & Install

bash 
git clone https://github.com/arnav-eluri/openswarm_hackathon_zeroday.git
cd openswarm_hackathon_zeroday
pip install -r requirements.txt
2

Configure Secrets

.env 
DISCORD_TOKEN=your_discord_bot_token_here
OPENROUTER_API_KEY=your_openrouter_api_key_here
3

Create Discord Bot

Head to the Discord Developer Portal, create a new app, enable Message Content Intent, and invite the bot to your server with bot + applications.commands scopes.

4

Launch

bash 
python discord_bot.py

Use /scan in any channel and paste a GitHub PR URL to start scanning.
